## Top 3 선정 근거

### 1위: 예약 생성 (9점)

**비즈니스 가치 (3점)**
- 캠핑장의 핵심 수익 기능
- 이 기능 없이는 사업 자체가 불가능

**사용 빈도 (3점)**
- 모든 고객이 반드시 거치는 핵심 경로
- 검색 → **예약** → 결제의 Critical Path

**실패 시 영향 (3점)**
- 예약 실패 = 매출 손실 직결
- 고객 이탈 → 경쟁 캠핑장으로 이동

---

### 2위: 기간별 사이트 검색 (9점)

**비즈니스 가치 (3점)**
- 예약의 진입점, 여기서 막히면 예약까지 도달 불가
- 연박 예약(고객 단가 높음)의 필수 기능

**사용 빈도 (3점)**
- 예약 전 모든 고객이 반드시 사용
- **검색** → 예약 → 결제의 Critical Path

**실패 시 영향 (3점)**
- 잘못된 검색 결과 → 예약 후 현장 충돌
- 고객 클레임 + 환불 + 신뢰 손상

---

### 3위: 예약 취소 (8점)

**비즈니스 가치 (3점)**
- 환불/정산과 직결되는 금전 기능
- 취소 정책은 고객 신뢰의 핵심

**사용 빈도 (2점)**
- 전체 고객 중 일부만 사용
- 하지만 사용 시 반드시 정확해야 함

**실패 시 영향 (3점)**
- 잘못된 취소 → 금전 클레임
- 환불 누락/오류 → 법적 분쟁 가능성

---

## 동점 처리 기준

**예약 생성 vs 기간별 검색 (둘 다 9점)**
- 예약 생성을 1위로 선정
- 이유: 검색은 실패해도 재검색 가능, 예약 실패는 즉시 매출 손실

**예약 조회 vs 예약 수정 vs 내 예약 조회 (모두 7점)**
- 예약 조회 > 예약 수정 > 내 예약 조회
- 이유: 조회는 모든 고객이 사용, 수정은 금전 영향, 내 예약은 대체 가능
